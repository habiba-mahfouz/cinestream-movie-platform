# CineStream 🎬

**A Full-Stack Movie Platform Web Application** 🎓

CineStream is a dynamic, full-stack movie streaming platform simulation. This project was collaboratively designed and developed from scratch by our team as our final project for the **Web Programming** course at Alexandria University.

Our primary goal was to build a robust application focusing on a seamless user experience, responsive UI design, and strong database architecture.

## 🚀 Key Features
- **User Authentication:** Secure login and registration system.
- **Dynamic Catalog:** Browse movies by categories (Action, Drama, Comedy).
- **My List:** Users can add and manage their favorite movies.
- **Responsive UI:** Custom-built UI using Vanilla CSS Box Model, Flexbox, and Grid (Includes Dark/Light Mode).

## 🛠️ Tech Stack
- **Frontend:** HTML5, CSS3, Vanilla JavaScript
- **Backend:** PHP (Procedural)
- **Database:** MySQL

## ⚙️ How to Run Locally
1. Clone this repository: `git clone https://github.com/habiba-mahfouz/cinestream-movie-platform.git`
2. Place the folder inside your XAMPP `htdocs` directory.
3. Import the `setup.sql` file into your MySQL database (via phpMyAdmin).
4. Update the `database.php` with your local database credentials if necessary.
5. Open `http://localhost/cinestream` in your web browser.
